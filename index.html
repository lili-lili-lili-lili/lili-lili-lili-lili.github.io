什么是React Hooks

React Hooks是React 16.7.0版本推出的新特性

React Hooks 要解决的问题是状态共享

大家都知道类组件和函数组件最大的区别就是一个是有状态(state)一个是无状态,下面给大家看一下hooks基本用法
js function App() { const [open, setOpen] = useState(false); return ( <> <Button type="primary" onClick={() => setOpen(true)}> Open Modal </Button> <Modal visible={open} onOk={() => setOpen(false)} onCancel={() => setOpen(false)} /> </> ); }
